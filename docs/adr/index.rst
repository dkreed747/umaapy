Architecture Decision Records
=============================

Conventions
-----------

- Naming: ``adr-XXXX-short-title.rst``
- Status values: ``Proposed``, ``Accepted``, ``Superseded``, ``Deprecated``
- Each ADR must include: context, decision, consequences, and follow-up actions
